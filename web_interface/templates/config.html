Параметры транскрипции
По умолчанию используется ваш HTTP-сервис транскрипции (Whisper). Режим Gemini обращается к API Google по тому же принципу, что SDK @google/genai в проекте okpd; на сервере приложения нужен пакет google-genai и ключ API.
Если ключ не задан в интерфейсе, для процесса анализатора может использоваться GEMINI_API_KEY в окружении. Прокси только для запросов к Gemini (остальной трафик без него): задайте в .env или окружении воркера GEMINI_PROXY_URL (полный URL, как в okpd) или GEMINI_PROXY в формате host:port:user:pass; при необходимости также читаются HTTPS_PROXY / HTTP_PROXY — они передаются только во внутренний HTTP-клиент Gemini, без смены глобального сетевого стека.
При включении используется стерео алгоритм из T‑Bank с анализом голосов; при выключении — стандартный моно режим.
При включении система пытается извлечь имя оператора из транскрипции (когда консультант представляется), затем использует таблицу привязки. При выключении сразу берет имя из таблицы EMPLOYEE_BY_EXTENSION.
По истечении срока исходные файлы (.mp3, .wav и др.) удаляются из папки записей. Транскрипции, разборы и отчёты не удаляются. Значение 0 — не удалять автоматически.
